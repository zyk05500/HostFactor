## 一些使用小技巧

买了VPS之后，如果你是新手，可能会遇到一些常见问题。这里分享几个我自己摸索出来的小技巧：

**关于SSH连接**：
HostFactor的VPS开通后会给你Root密码。首次登录建议立刻改密码，并且最好配置SSH密钥登录（更安全）。如果你用的是Windows系统，可以用PuTTY或者新版Windows 10/11自带的SSH客户端。

**关于防火墙**：
默认情况下系统可能没开防火墙，记得手动配置iptables或firewalld。基本原则就是只开放必需的端口（比如22、80、443），其他全关掉。

**关于备份**：
虽然HostFactor提供每日自动备份，但我还是建议你自己也做一份异地备份。可以用rsync定时把重要数据同步到另一个位置，或者用rclone备份到云存储（比如Google Drive、Backblaze B2等）。

**关于监控**：
可以装个简单的监控工具，比如Netdata或者Prometheus + Grafana，实时查看CPU、内存、磁盘、网络使用情况。这样出问题能第一时间发现。

**关于性能优化**：
如果你用VPS跑网站，建议装个缓存系统（比如Redis或Memcached），数据库记得调优（MySQL的话可以用mysqltuner这个脚本），Web服务器配置好Gzip压缩和HTTP/2。这些优化做下来，性能能提升一大截。
